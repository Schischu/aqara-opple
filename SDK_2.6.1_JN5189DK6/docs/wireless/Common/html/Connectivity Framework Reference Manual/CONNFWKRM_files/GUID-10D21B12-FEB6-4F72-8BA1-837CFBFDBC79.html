Deep power-down mode

If full power down enabled and set cPWR_EnableDeepSleepMode_4 to 1, system can support deep power-down mode. Calling PWR_ChangeDeepSleepMode(4) sets the system in the deep power-down mode.

Deep power-down mode shuts down virtually all on-chip power consumption, but require a longer wake-up time.

Reset pin or a IO event can cause the wake-up.
