Using sniffer binary with Ubiqua Protocol Analyser
--------------------------------------------------
1. Program a module or USB dongle with the sniffer binary Sniffer_1000000baud_8N1_NoFlowControl.bin
2. In Ubiqua:
     a. Under menu "Device" select "Add Device"
     b. Under "Available Devices", select the COM port for the module or USB dongle
     c. Under "Application", select "NXP JN51xx/Rainforest Toucan"
     d. Under "Confuiguration", select Baud Rate 1000000, Data Bits 8, Stop Bits 1, Parity None, Flow Control None
     e. Click "Add Device"
